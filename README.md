# 全自动策略选股
交易日每天18点自动按指定规则筛选符合条件的A股股票
# 命名规则
- 选股结果：stock_chose_[股则]_[日期].csv
- 个股数据：stock_[adjust]_[股票代码].csv

## 数据存储目录/data
## 运行规则如下
 - rule1:100-200亿流通市值&15天内持续收盘价在250日均线上下
 - rule2:200-500亿流通市值&15天内持续收盘价在250日均线上下
